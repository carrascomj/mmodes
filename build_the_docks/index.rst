Welcome to MMODES's documentation!
==================================

MMODES is a package built on top of `COBRApy <https://opencobra.github.io/cobrapy/>`_
and `SciPy <https://www.scipy.org/>`_ to allow fast set up of dynamic simulations with GEM models (dFBA),
isolated or in a consortium.
